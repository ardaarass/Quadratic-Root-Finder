Program Hakkında:

Bu program (ax² + bx + c) şeklindeki ikinci dereceden bir denklemin a,b,c değerlerini kullnarak delta formulü aracılığıyla kökün olup olmadığını bulur,varsa kökleri hesaplar.

Nasıl Çalışır?

1)Sırasıyla a,b,c değerlerini giriniz.

2)İkinci dereceden denkleminiz için kök olup olmadığı ekrana yazdırılır.

3)Kök veya kökler varsa hesaplanarak ekranda görüntülenir.
